# NFR X-Ray Checklist

Use this checklist as a guide. Do not treat it as exhaustive.

## Security

- Authentication and authorization.
- Tenant isolation.
- Input validation.
- File upload restrictions.
- Secrets management.
- Dependency risk.
- Abuse and rate limiting.

## Privacy And Compliance

- PII minimization.
- Retention and deletion.
- Audit trails.
- Data residency.
- Encryption in transit and at rest.
- Access logs and approval history.

## Performance And Scalability

- File size limits.
- Synchronous vs asynchronous processing.
- Queueing and backpressure.
- Database growth.
- Pagination.
- Bulk upload behavior.

## Resilience

- External service timeouts.
- Retries and circuit breakers.
- Idempotency.
- Partial failure handling.
- Disaster recovery.

## Observability And Operability

- Structured logs.
- Metrics.
- Tracing.
- Alerting.
- Runbooks.
- Admin support tools.

## Accessibility

- Keyboard navigation.
- Labels and names.
- Color contrast.
- Focus states.
- Error messages.
- Screen reader behavior.
