# Starter App

This is a deliberately rough demo app for the NFR X-Ray exercise.

It is not intended to be production-ready. Your task is to identify what would block or constrain production use and recommend a practical path to enterprise readiness.

## Run Locally

```bash
npm install
npm start
```

Then open `http://localhost:3000`.

## Run Tests

```bash
npm test
```

The app code is intentionally demo-grade. Use the modules and tests as evidence while building your NFR report.
