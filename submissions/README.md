# Submissions

Learners can place written artifacts here when an exercise asks for a report, review, or design note.

Recommended structure:

```text
submissions/<github-handle>/ex-1/review.md
submissions/<github-handle>/ex-2/nfr-xray.md
submissions/<github-handle>/ex-3/refactor-notes.md
```

It is also valid to change exercise code in your fork when the exercise requires implementation.
