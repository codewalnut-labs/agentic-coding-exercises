# Exercise Rules

These exercises are designed to test how well a participant can direct, verify, and improve work done with an AI coding agent.

## Allowed

- Use an AI coding agent.
- Ask the agent to inspect code, write tests, refactor, and generate reports.
- Edit the agent output manually.
- Add tests and documentation.
- Mention assumptions and tradeoffs.

## Expected

- Verify the agent's work.
- Keep changes scoped to the exercise.
- Preserve existing behavior unless the exercise asks you to change it.
- Include evidence: test output, reasoning, review notes, screenshots, or command logs.

## Not Expected

- Do not modify unrelated exercises.
- Do not rewrite the repository structure.
- Do not submit generated text without reviewing it.
- Do not include secrets, private data, or real customer information.
